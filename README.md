# 🌾 AI-Powered Crop Recommendation System

This repository contains the source code and trained machine learning model for our precision agriculture capstone project. Our system removes farming guesswork by matching soil chemistry and weather patterns to the ideal crop type.

---

## 🚀 Project Overview
* **Domain:** Data Science / Machine Learning
* **Core Algorithm:** Random Forest Classifier
* **Dataset Size:** 2,200 unique records across 22 distinct crop classes
* **Target Success Rate:** 98.2% Accuracy

---

## 🛠️ Repository Structure
* `app.py` - The user interface code designed for the deployment stage.
* `crop_recommendation_model.pkl` - The trained self-contained AI model.
* `requirements.txt` - The configuration file listing required software libraries.
